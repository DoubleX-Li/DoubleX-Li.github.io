Windows中删除路径太长目录及文件

当前目录下shift+鼠标右键在此处打开命令窗口

robocopy empty_dir will_delete_dir /purge

  • empty_dir 新建的空白目录
  • will_delete_dir 要删除的目录
  • 注意中间的空格

注意此方法将删除目标路径下所有文件，慎用！！！

07/22/2015

查看占用端口的进程并终止

在使用Tomcat的过程中总有端口被占用的情况，此时需要找到占用端口的进程并杀掉

windows

查看所有的端口占用情况

netstat -ano

会返回：

协议 本地地址 外部地址 状态 PID

查看指定的端口占用情况

netstat -ano|findstr "端口号"

会返回：

协议 本地地址 外部地址 状态 PID

查看PID对应的进程

tasklist|findstr "PID"

会返回：

映像名称 PID 会话名 会话# 内存使用

杀死该进程

taskkill /T /F /PID PID

Linux

查看端口（如8080端口）

netstat -lnp|grep 8080

会返回：

tcp6 0 0 :::8080 :::* LISTEN 5263/java

杀死占用端口的进程

kill -9 5263
